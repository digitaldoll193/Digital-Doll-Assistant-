## How to run it locally

1. Install Node.js.
2. Open this folder in a terminal.
3. Run:

npm install

4. Copy `.env.example` to `.env`.
5. Add your OpenAI API key in `.env`.
6. Run:

npm start

7. Open:

http://localhost:3000

## Wix install code

After the backend is hosted online, use:

<script src="YOUR-LIVE-BACKEND-URL/widget.js" data-api="YOUR-LIVE-BACKEND-URL"></script>

Paste that into Wix custom code or HTML embed.
